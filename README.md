## Description  
A simple and efficient Loan EMI Calculator built using HTML, CSS and JavaScript. This application helps users calculate their monthly EMI based on loan amount, interest rate and loan tenure.

## Features  
- Input fields for Loan Amount, Interest Rate and Tenure  
- Automatic EMI calculation using JavaScript  
- Instant display of Monthly EMI, Total Payment and Total Interest  
- Clean and responsive UI  
- Works fully on the frontend without any backend  

## Tech Stack  
- HTML  
- CSS  
- JavaScript  

## How to Use  
1. Clone or download the repository  
2. Open `index.html` in a web browser  
3. Enter loan details and get EMI instantly

JavaScript calculations, form handling, DOM manipulation, responsive UI design
